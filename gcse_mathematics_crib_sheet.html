GCSE Mathematics — Core Topics Crib Sheet

Foundation + Higher Tier · Formulas · Exam Board Coverage
Number & Calculations Foundation focus; Higher assumes mastery
Integers, decimals, fractions
Order of operations: BIDMAS/PEMDAS (Brackets, Indices, Division/Multiplication, Addition/Subtraction)
eg: 3 + 4 × 2² = 3 + 4 × 4 = 3 + 16 = 19
All boards
Percentages of amounts
Percentage = (Part ÷ Whole) × 100
eg: 15% of £200 = 0.15 × 200 = £30 All boards
Percentage change
% change = (New − Old) ÷ Old × 100
If price rises £10→£12: (2÷10)×100 = 20% increase All boards
Ratios & scaling
Divide both by common factor to simplify.
Ratio 15:20 = 3:4 (divide by 5)
Scale recipes, maps, ingredients All boards
Surds Higher Tier
√36 = 6; √12 = √(4×3) = 2√3 (simplify)
Rationalise: multiply by conjugate. Eg: 1/√2 = √2/2 Higher Tier
Common error: √(9+16) ≠ 3+4. Always simplify inside first: √25 = 5 ✓
Algebra — Foundations All boards
Simplify expressions
3x + 2x − 4x = x; 2a × 3b = 6ab; (2x)² = 4x²
Collect like terms; apply index laws All boards
Expand brackets
2(3x + 5) = 6x + 10
(x+3)(x−2) = x² − 2x + 3x − 6 = x² + x − 6 (FOIL)
All boards
Factorise
6x + 9 = 3(2x + 3) — find common factor
x² + 5x + 6 = (x+2)(x+3) — find two numbers that multiply to 6 and add to 5
All boards
Solve linear equations
3x + 7 = 16 → 3x = 9 → x = 3
Do same operation to both sides. Check: 3(3)+7 = 16 ✓ All boards
Solve quadratics Higher Tier
x² + 5x + 6 = 0 → (x+2)(x+3) = 0 → x = −2 or x = −3
Or use formula: x = [−b ± √(b²−4ac)] ÷ 2a Higher Tier
Sequences & Functions Higher: essential
Arithmetic sequences
nth term = a + (n−1)d, where a = first term, d = common difference
eg: 3, 7, 11, 15... → a=3, d=4; 10th term = 3+(10−1)×4 = 39 Higher Tier
Linear functions
y = mx + c, where m = gradient, c = y-intercept
Gradient = rise/run. Eg: line through (0,2) and (1,5): m = 3/1 = 3; equation: y = 3x + 2 Higher Tier
Quadratic graphs
y = ax² + bx + c
Opens upward (a>0) or downward (a<0). Vertex at x = −b/2a. Symmetrical parabola Higher Tier
Geometry — Angles & Shapes All boards
Angle facts
Angles on straight line = 180°; around a point = 360°; vertically opposite angles are equal
Interior angle of polygon = (n−2)×180 ÷ n, where n = number of sides
All boards
Triangle properties
Angles sum to 180°. Exterior angle = sum of two opposite interior angles. Pythagoras (Right triangles):
a² + b² = c²
All boards
Circle facts Higher
Angle in semicircle = 90°. Angles subtended from same arc are equal. Tangent perpendicular to radius Higher Tier
Circle Theorems
Higher Tier
Congruence & similarity
Same shape, same size = congruent. Same shape, different size = similar. Scale factor k: side lengths scale by k, areas by k², volumes by k³ All boards
Trigonometry Higher Tier; some Foundation intro
SOHCAHTOA (right triangles)
sin(θ) = Opposite/Hypotenuse; cos(θ) = Adjacent/Hypotenuse; tan(θ) = Opposite/Adjacent
To find side: side = known × sin/cos/tan(angle). To find angle: angle = sin⁻¹(opp/hyp) etc Higher Tier
Sine rule Higher
a/sin(A) = b/sin(B) = c/sin(C)
Use when you know angle-side pair or two angles Higher Tier
Cosine rule Higher
a² = b² + c² − 2bc cos(A)
Use when you know all sides or two sides + included angle Higher Tier
Area of triangle
Area = ½ × base × height
Or with angle:
Area = ½ab sin(C)
All boards
Area & Volume All boards
2D areas
Rectangle: l × w; Triangle: ½bh; Circle: πr²; Trapezium: ½(a+b)h
All boards
3D volumes
Cuboid: l × w × h; Cylinder: πr²h; Cone: ⅓πr²h; Sphere: ⁴⁄₃πr³
Prism volume = base area × length All boards
Surface area
Lateral + base areas.
Cylinder SA = 2πrh + 2πr² (curved + 2 circles)
Higher Tier typically
Density, mass, volume
Density = Mass ÷ Volume
eg: Object 500g, volume 100 cm³ → density = 5 g/cm³ All boards
Statistics & Probability All boards
Averages
Mean = sum of all values ÷ count
Median = middle value when ordered. Mode = most frequent. Range = highest − lowest All boards
Grouped data
Use class midpoints to estimate mean.
Mean ≈ (Σ midpoint × frequency) ÷ total frequency
All boards
Probability
P(event) = number of favorable outcomes ÷ total outcomes
Independent events: P(A and B) = P(A) × P(B).
Coin twice: P(HH) = ½ × ½ = ¼
All boards
Conditional probability Higher
P(A|B) = P(A and B) ÷ P(B)
Probability of A given B has occurred Higher Tier
Normal distribution Higher
68% within ±1σ (std dev); 95% within ±2σ. Use z-score: z = (x−μ)/σ to standardize Higher Tier
Vectors & Transformations Higher Tier
Vectors notation
v = (3, −2) means 3 units right, 2 units down
Magnitude: |v| = √(3² + (−2)²) = √13 Higher Tier
Vector operations
(a,b) + (c,d) = (a+c, b+d); k(a,b) = (ka, kb) (scale)
Higher Tier
Translations, rotations, reflections
Translation by vector (a,b): each point moves a right, b up. Rotation: specify center, angle, direction (clockwise/counterclockwise). Reflection: specify mirror line All boards
Enlargement
Scale factor k from center. All lengths multiply by k; angles unchanged. Negative k = rotation 180° + enlargement All boards
Calculus Basics Higher Tier only
Differentiation (dx)
If y = xⁿ, then dy/dx = nxⁿ⁻¹
eg: y = x³ → dy/dx = 3x². Gradient of tangent at point. Higher Tier
Turning points
Set dy/dx = 0 to find x-coords of maxima/minima. Check second derivative: d²y/dx² > 0 = minimum, < 0 = maximum Higher Tier
Integration (∫)
∫xⁿ dx = [xⁿ⁺¹ ÷ (n+1)] + C (indefinite integral)
Reverse of differentiation. Area under curve (definite) Higher Tier
✓ Always add +C for indefinite integrals. ✓ Check dimension consistency (units)
