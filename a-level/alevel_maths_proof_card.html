The Four Types of Proof

Method 1
Direct Proof
Start from known facts,
reach the conclusion logically.
Most common type.
Method 2
By Exhaustion
Check every possible case.
Only when there are
finitely many cases.
Method 3
Counter-Example
Find ONE case where
the statement fails.
Quickest way to disprove.
Method 4
Contradiction
Assume the opposite is true,
show it leads to
a contradiction.

Direct Proof — The Template

Template: Direct Proof (Deduction)
1
State your variable "Let n be an integer"
2
Express the thing you're proving in terms of n
3
Use algebra to reach the required form
4
State the conclusion explicitly "Therefore ... is divisible by ..."
Example 1: Prove that the sum of two consecutive odd numbers is divisible by 4
1
Let the two consecutive odd numbers be (2n − 1) and (2n + 1) for some integer n
2
Add them together: (2n − 1) + (2n + 1) = 4n
3
4n = 4 × n, which is a multiple of 4 for any integer n
Therefore the sum of two consecutive odd numbers is always divisible by 4 □
Example 2: Prove that (n+1)² − n² = 2n + 1
1
Start from the left-hand side (LHS): (n+1)² − n²
2
Expand (n+1)²: = n² + 2n + 1 − n²
3
Simplify: = 2n + 1 = RHS
LHS = RHS, therefore (n+1)² − n² = 2n + 1 □

Proof by Exhaustion — The Template

Template: Proof by Exhaustion
1
List ALL possible cases
2
Check each one satisfies the statement
3
State the conclusion "All cases have been checked, therefore the statement is true"
Example: Prove that n² + n is even for all integers n between 1 and 4
1
n = 1:  1² + 1 = 2 2 is even ✓
2
n = 2:  2² + 2 = 6 6 is even ✓
3
n = 3:  3² + 3 = 12 12 is even ✓
4
n = 4:  4² + 4 = 20 20 is even ✓
All cases checked. n² + n is even for n = 1, 2, 3, 4 □

Disproof by Counter-Example — The Template

Template: Disproof by Counter-Example
1
State your chosen value "Consider n = ..."
2
Show the statement fails for this value
3
State the conclusion "This is a counter-example, therefore the statement is false"
Example: Disprove — "n² > 3n for all positive integers n"
1
Consider n = 1: n² = 1² = 1
2
Compare: is 1 > 3(1)? 1 > 3? — No!
n = 1 is a counter-example. The statement is false. □

Proof by Contradiction — The Template

Template: Proof by Contradiction
1
Assume the opposite "Assume, for contradiction, that..."
2
Follow the logical consequences of this assumption
3
Reach a statement that contradicts a known fact
4
State the conclusion "This is a contradiction, therefore our assumption was wrong, so..."
Example 1: Prove that √2 is irrational
1
Assume, for contradiction, that √2 IS rational. Then √2 = a/b where a, b are integers with no common factors
2
Square both sides: 2 = a²/b², so a² = 2b² a² is even, so a must be even. Write a = 2k.
3
Substitute: (2k)² = 2b² ⇒ 4k² = 2b² ⇒ b² = 2k² b² is even, so b must also be even.
4
But if both a and b are even, they share a common factor of 2. This contradicts our assumption that a/b has no common factors!
Contradiction! Therefore √2 is irrational □
Example 2: Prove there are infinitely many primes
1
Assume, for contradiction, that there are finitely many primes. List them all: p&sub1;, p&sub2;, p&sub3;, ..., pₙ
2
Construct the number N = p&sub1; × p&sub2; × p&sub3; × ... × pₙ + 1
3
N is not divisible by any prime in our list (dividing by any pᵢ leaves remainder 1). So N is either prime itself, or has a prime factor not in our list.
4
Either way, there exists a prime not in our "complete" list. This contradicts our assumption that we listed ALL primes!
Contradiction! Therefore there are infinitely many primes □

Trig Identity Proofs — The Method

The Rules for Trig Proofs
1
Start from ONE SIDE (usually the more complex side)
2
Use known identities: sin²θ + cos²θ ≡ 1   •   tanθ ≡ sinθ / cosθ
3
Manipulate algebraically until you reach the OTHER SIDE
!
NEVER work from both sides towards the middle — that is not a valid proof
Example: Prove that (1 − cos²θ) / sinθ ≡ sinθ
1
Start from the LHS (the more complex side): (1 − cos²θ) / sinθ
2
Use the identity sin²θ + cos²θ ≡ 1, so 1 − cos²θ = sin²θ: = sin²θ / sinθ
3
Cancel one sinθ: = sinθ = RHS
LHS = RHS — identity proven □

Common Mistakes

Not starting with a clear assumption or statement of variables
Working from both sides in a trig proof — you must start from one side only
Using specific numbers instead of general variables in a direct proof — "it works for n = 3" is not proof
Forgetting to state the conclusion — always write "therefore..."
"Showing it works for a few cases" is NOT proof (unless it's exhaustion covering ALL cases)
Prove it? → Direct proof
Disprove it? → One counter-example
Seems impossible? → Assume the opposite
Finite cases? → Check them all
State. Manipulate. Conclude. Always say why. Always end with a box.
