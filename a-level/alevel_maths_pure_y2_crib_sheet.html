Further Trigonometry Ch 6–7
Reciprocal trig functions
secθ = 1/cosθ  ·  cosecθ = 1/sinθ  ·  cotθ = 1/tanθ = cosθ/sinθ
sec is undefined where cosθ = 0, cosec where sinθ = 0, cot where tanθ = 0. Graphs are reflections/reciprocals of the standard trig curves with vertical asymptotes at the zeros.
Pythagorean identities
sin²θ + cos²θ ≡ 1
sec²θ ≡ 1 + tan²θ   (divide by cos²θ)
cosec²θ ≡ 1 + cot²θ   (divide by sin²θ)
The second and third identities are obtained by dividing the first by cos²θ and sin²θ respectively.
Compound angle formulas
sin(A ± B) = sinA cosB ± cosA sinB
cos(A ± B) = cosA cosB ∓ sinA sinB
tan(A ± B) = (tanA ± tanB) / (1 ∓ tanA tanB)
Double angle formulas
sin2A = 2sinA cosA
cos2A = cos²A − sin²A = 2cos²A − 1 = 1 − 2sin²A
tan2A = 2tanA / (1 − tan²A)
The cos2A forms rearrange to give half-angle substitutions for integration:
cos²A = ½(1 + cos2A)
sin²A = ½(1 − cos2A)
R-formula (harmonic form)
a sinθ + b cosθ = R sin(θ + α)
a cosθ + b sinθ = R cos(θ − α)
where R = √(a² + b²), tanα = b/a
Used to solve equations of the form a sinθ + b cosθ = c, and to find max/min values of trig expressions.
Example3sinθ + 4cosθ = R sin(θ+α) → R = √(9+16) = 5, tanα = 4/3 → α ≈ 53.1°
Small angle approximations
sinθ ≈ θ   ·   cosθ ≈ 1 − θ²/2   ·   tanθ ≈ θ
Valid when θ is small and measured in radians. Used for limits and approximations.
cos(A ± B): the sign FLIPS — cos(A+B) uses minus, cos(A−B) uses plus. The opposite of what you might expect.
For R-formula questions: always state R and α, then rewrite the expression before solving. Max value of R sin(θ+α) is R, occurring when sin(θ+α) = 1.
Further Differentiation Ch 9–10
Chain rule
dy/dx = dy/du × du/dx
For composite functions y = f(g(x)): differentiate the outer function, multiply by the derivative of the inner.
Exampley = (3x+1)⁵ → dy/dx = 5(3x+1)⁴ × 3 = 15(3x+1)⁴
Product rule
d(uv)/dx = u(dv/dx) + v(du/dx)
Exampley = x²sinx → dy/dx = x²cosx + 2x sinx
Quotient rule
d(u/v)/dx = [v(du/dx) − u(dv/dx)] / v²
Remember: "low d-high minus high d-low, over the square of what's below."
Trig derivatives
d/dx(sinx) = cosx
d/dx(cosx) = −sinx
d/dx(tanx) = sec²x
Combined with chain rule: d/dx(sin(kx)) = k cos(kx), etc.
Exponential & log
d/dx(eˣ) = eˣ  ·  d/dx(e^(kx)) = ke^(kx)
d/dx(ln x) = 1/x  ·  d/dx(ln(f(x))) = f′(x)/f(x)
Implicit differentiation
Differentiate both sides with respect to x. When differentiating a y-term, multiply by dy/dx (chain rule).
d/dx(y²) = 2y · dy/dx
d/dx(xy) = x · dy/dx + y  (product rule)
Then rearrange to isolate dy/dx.
Parametric differentiation
dy/dx = (dy/dt) / (dx/dt)
For curves defined as x = f(t), y = g(t). Differentiate each with respect to t, then divide.
Connected rates
dA/dt = dA/dr × dr/dt
Chain rule applied to real-world rates. Identify which rate you know, which you want, and find the connecting variable.
Quotient rule: the order matters — it is v(du/dx) MINUS u(dv/dx), not the other way round. Getting this backwards is the most common mistake.
For implicit differentiation: collect all dy/dx terms on one side, factor out dy/dx, then divide. Treat dy/dx as a single object you are solving for.
Differential Equations Ch 11
Separable variables
If dy/dx = f(x)g(y), separate and integrate:
∫ 1/g(y) dy = ∫ f(x) dx
Exampledy/dx = xy → ∫(1/y)dy = ∫x dx → ln|y| = x²/2 + c → y = Ae^(x²/2)
Particular solutions
Use boundary/initial conditions to find the constant of integration. Substitute the given point (x₀, y₀) into the general solution and solve for c (or A).
Modelling
dP/dt = kP → P = P₀eᵏᵗ  (exponential growth/decay)
dT/dt = −k(T − T₀) → Newton's cooling
Set up the differential equation from the word problem, solve by separation, then apply conditions.
When separating variables, do not forget to integrate BOTH sides. And always include +c on one side (not both).
After solving, always substitute back to check your solution satisfies the original DE and the boundary conditions.
Further Integration Ch 11
Standard results
∫ eˣ dx = eˣ + c  ·  ∫ e^(kx) dx = (1/k)e^(kx) + c
∫ 1/x dx = ln|x| + c
∫ sinx dx = −cosx + c  ·  ∫ cosx dx = sinx + c
∫ sec²x dx = tanx + c
Integration by substitution
Replace a complicated part with u, find du/dx, substitute everything including dx:
∫ f(g(x)) · g′(x) dx → let u = g(x), du = g′(x)dx → ∫ f(u) du
For definite integrals: change the limits to u-values, or convert back to x at the end.
Example∫ 2x(x²+1)³ dx, let u = x²+1, du = 2x dx → ∫ u³ du = u⁴/4 + c = (x²+1)⁴/4 + c
Integration by parts
∫ u dv = uv − ∫ v du
Choose u using LIATE priority: Logs, Inverse trig, Algebraic, Trig, Exponential.
Example∫ x eˣ dx: u = x, dv = eˣ dx → du = dx, v = eˣ → xeˣ − ∫eˣ dx = xeˣ − eˣ + c
Sometimes need to apply twice (e.g. ∫ x²eˣ dx) or use the "cycling" trick for ∫ eˣsinx dx.
Partial fractions
Split rational functions before integrating:
A/(x−a) integrates to A ln|x−a| + c
A/(x−a)² integrates to −A/(x−a) + c
Example∫ (3x+5)/((x+1)(x+2)) dx → ∫ [2/(x+1) + 1/(x+2)] dx = 2ln|x+1| + ln|x+2| + c
Trapezium rule
∫ₐᵇ y dx ≈ h/2 [y₀ + 2(y₁+y₂+...+yₙ₋₁) + yₙ]
where h = (b−a)/n
More strips (larger n) gives a better approximation. Overestimates for convex curves, underestimates for concave.
Parametric integration
∫ y dx = ∫ y · (dx/dt) dt
Substitute x = f(t), y = g(t), and change limits to t-values. The dx becomes (dx/dt)dt.
By parts: choosing the wrong u and dv makes the integral harder, not easier. If it gets more complicated, swap your choice.
For ∫ sin²x dx or ∫ cos²x dx, use the double angle identity: sin²x = ½(1 − cos2x), cos²x = ½(1 + cos2x).
Sequences & Series Ch 3–4
Arithmetic sequences
uₙ = a + (n−1)d
Sₙ = n/2 · (2a + (n−1)d) = n/2 · (a + l)
a = first term, d = common difference, l = last term. Sum formula: average of first and last, times n.
Geometric sequences
uₙ = arⁿ⁻¹
Sₙ = a(1 − rⁿ) / (1 − r)   (r ≠ 1)
a = first term, r = common ratio. Find r by dividing consecutive terms: r = u₂/u₁.
Sum to infinity
S∞ = a / (1 − r)   when |r| < 1
Converges only when |r| < 1. If |r| ≥ 1, the series diverges and has no finite sum.
Binomial expansion (rational n)
(1+x)ⁿ = 1 + nx + n(n−1)x²/2! + n(n−1)(n−2)x³/3! + ...
Valid when |x| < 1
For (a+bx)ⁿ: factor out aⁿ first → aⁿ(1 + bx/a)ⁿ, then expand with |bx/a| < 1.
Example(1+x)⁻¹ = 1 − x + x² − x³ + ...   (|x| < 1)
Sigma notation
∑ᵣ₌₁ⁿ f(r) = f(1) + f(2) + ... + f(n)
Useful sums: ∑r = n(n+1)/2, ∑r² = n(n+1)(2n+1)/6.
Binomial expansion for rational n is an INFINITE series, not finite. You must state the validity condition |x| < 1 (or equivalent) — marks are lost without it.
For geometric series: always check |r| < 1 before using S∞. In proof questions, show that rⁿ → 0 as n → ∞.
Numerical Methods Ch 8
Sign change method
If f(a) and f(b) have opposite signs AND f is continuous on [a,b], then there is a root in (a,b).
f(a) > 0 and f(b) < 0 (or vice versa) → root exists in (a, b)
Iteration
Rearrange f(x) = 0 into x = g(x), then iterate:
xₙ₊₁ = g(xₙ)
Converges when |g′(x)| < 1 near the root. Different rearrangements may converge to different roots or diverge.
Examplex³ = 5x + 1 → x = (5x+1)^(1/3), start x₀ = 2: x₁ = ∛11 ≈ 2.224...
Newton-Raphson
xₙ₊₁ = xₙ − f(xₙ) / f′(xₙ)
Usually converges much faster than simple iteration. Can fail if f′(xₙ) ≈ 0 or if the starting point is poor (near a turning point).
Trapezium rule
∫ₐᵇ y dx ≈ h/2 [y₀ + 2(y₁+...+yₙ₋₁) + yₙ]
h = (b−a)/n
Increasing n (more strips) improves accuracy. State whether the estimate is an overestimate or underestimate by considering the curve's concavity.
Newton-Raphson can diverge or oscillate if the starting value is near a stationary point — always check that f′(xₙ) is not close to zero.
In exams, show each iteration step and give values to the required decimal places. State convergence by showing two consecutive iterates agree to the required accuracy.
Vectors (3D) Ch 12
3D vectors
Position vectors in 3D: r = xi + yj + zk or as column vectors.
|a| = √(x² + y² + z²)
Unit vector: â = a / |a|. Direction ratios give the relative components.
Scalar (dot) product
a · b = a₁b₁ + a₂b₂ + a₃b₃ = |a||b|cosθ
If a · b = 0, the vectors are perpendicular. Rearrange to find the angle:
cosθ = (a · b) / (|a||b|)
Angle between vectors
Use the scalar product formula. Always give the acute angle unless told otherwise — if cosθ is negative, the acute angle is 180° − θ.
Vector equation of a line
r = a + λd
a = position vector of a known point on the line, d = direction vector, λ = parameter.
ExampleLine through (1,2,3) in direction (4,−1,2): r = (1,2,3) + λ(4,−1,2)
To find if a point lies on the line: substitute and check all three components give the same λ.
When finding angles between lines, use the direction vectors — not position vectors. Take |cosθ| if you need the acute angle.
To find the intersection of two lines in 3D, set them equal and solve the simultaneous equations. Check the third equation for consistency — in 3D, lines may be skew (non-intersecting and non-parallel).
