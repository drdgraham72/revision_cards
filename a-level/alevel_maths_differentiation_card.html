Answer Yes or No. Three questions maximum.
The Chain Rule
dy/dx = dy/du × du/dx
"Differentiate the outside, keep the inside, multiply by the derivative of the inside"
1 Identify the inner function u and the outer function y
2 Differentiate the outer function (keeping inner as u)
3 Differentiate the inner function
4 Multiply them together

Example 1: sin(3x)

u = 3x,   y = sin(u)
dy/du = cos(u),   du/dx = 3
dy/dx = cos(3x) × 3 = 3cos(3x) ✓

Example 2: (2x + 1)&sup5;

u = 2x + 1,   y = u&sup5;
dy/du = 5u&sup4;,   du/dx = 2
dy/dx = 5(2x+1)&sup4; × 2 = 10(2x+1)&sup4; ✓

Example 3: e^(x²)

u = x²,   y = eu
dy/du = eu,   du/dx = 2x
dy/dx = ex² × 2x = 2x·ex² ✓
The Product Rule
d(uv)/dx = u(dv/dx) + v(du/dx)
"first × d(second) + second × d(first)"
1 Label u and v (the two parts being multiplied)
2 Find du/dx and dv/dx separately
3 Apply the formula: u·dv/dx + v·du/dx

Example 1: x² · sin(x)

u = x²,   v = sin(x)
du/dx = 2x,   dv/dx = cos(x)
dy/dx = x²·cos(x) + sin(x)·2x
= x²cos(x) + 2x·sin(x) ✓

Example 2: ex · ln(x)

u = ex,   v = ln(x)
du/dx = ex,   dv/dx = 1/x
dy/dx = ex·(1/x) + ln(x)·ex
= ex(1/x + ln(x)) ✓
The Quotient Rule
d(u/v)/dx = [v(du/dx) − u(dv/dx)] / v²
"lo d-hi minus hi d-lo, over lo-lo"
Or rewrite u/v as u·v−1 and use product + chain rule instead.
1 Label u (top) and v (bottom)
2 Find du/dx and dv/dx separately
3 Apply: (v·du/dx − u·dv/dx) / v²

Example 1: sin(x) / x²

u = sin(x),   v = x²
du/dx = cos(x),   dv/dx = 2x
dy/dx = [x²·cos(x) − sin(x)·2x] / x&sup4;
= [x·cos(x) − 2sin(x)] / x³ ✓

Example 2: (x+1) / (x−1)

u = x+1,   v = x−1
du/dx = 1,   dv/dx = 1
dy/dx = [(x−1)·1 − (x+1)·1] / (x−1)²
= [x−1−x−1] / (x−1)² = −2/(x−1)² ✓

Common Mistakes

  • Forgetting the chain rule on composite functions:
    d/dx[sin(3x)] = cos(3x) → 3cos(3x)
  • Sign errors with cos → −sin:
    d/dx[cos(x)] = sin(x) → −sin(x)
  • Mixing up product and quotient rule formulas: product has a +, quotient has a − (and the square on the bottom).
  • Not simplifying the answer: always factorise and cancel where possible — marks are often awarded for the simplified form.
