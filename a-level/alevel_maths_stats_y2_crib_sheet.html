Normal Distribution Ch 3
N(μ, σ²) notation
X ~ N(μ, σ²) means X is normally distributed with mean μ and variance σ². The distribution is symmetric, bell-shaped, centred on μ. Total area under the curve = 1.
About 68% within μ ± σ
About 95% within μ ± 2σ
About 99.7% within μ ± 3σ
Standardising
Convert X ~ N(μ, σ²) to Z ~ N(0, 1) using:
Z = (X − μ) / σ
This lets you use the standard normal table. Z tells you how many standard deviations X is from the mean.
Using tables
Tables give P(Z < z) for positive z values. For negative z, use symmetry:
P(Z < −a) = 1 − P(Z < a)
P(Z > a) = 1 − P(Z < a)
For P(a < Z < b): compute P(Z < b) − P(Z < a).
Inverse normal
Given P(X < k) = p, find k. Steps:
1. Look up p in the body of the table to find z
2. Use k = μ + zσ
If P(X < k) = p, find z such that Φ(z) = p
Then k = μ + zσ
Finding μ or σ
Set up simultaneous equations when given two probability conditions. Standardise each to get two equations in μ and σ, then solve.
ExampleP(X < 20) = 0.2 → (20 − μ)/σ = −0.8416
P(X < 50) = 0.9 → (50 − μ)/σ = 1.2816
Solve simultaneously for μ and σ.
The parameter in N(μ, σ²) is the VARIANCE, not the standard deviation. If given σ = 3, then X ~ N(μ, 9). When standardising, divide by σ (standard deviation), not σ².
Sketch the normal curve, shade the required area, and mark z-values before using tables. This prevents sign errors and helps you see whether to subtract from 1.
Normal Approximation to Binomial Ch 4
When to use
If X ~ B(n, p) and n is large with np > 5 and n(1−p) > 5, then X can be approximated by:
X ≈ N(np, np(1−p))
Mean = np, Variance = np(1−p)
Continuity correction
The binomial is discrete; the normal is continuous. Apply a continuity correction of ±0.5:
P(X ≤ n) → P(Y < n + 0.5)
P(X < n) → P(Y < n − 0.5)
P(X ≥ n) → P(Y > n − 0.5)
P(X > n) → P(Y > n + 0.5)
P(X = n) → P(n − 0.5 < Y < n + 0.5)
Method
1. Verify conditions: n large, np > 5, n(1−p) > 5
2. Calculate mean μ = np and variance σ² = np(1−p)
3. Apply continuity correction
4. Standardise: Z = (X − μ)/σ
5. Use normal tables
Forgetting the continuity correction is the single most common error on this topic. Always adjust by 0.5 in the direction that makes the interval wider (includes the boundary value).
Think: "≤ means I want to include that value, so I go 0.5 beyond it" — the correction always moves the boundary away from the mean to capture the discrete value.
Conditional Probability (Advanced) Ch 2
Formula
P(A|B) = P(A ∩ B) / P(B)
Rearranges to: P(A ∩ B) = P(A|B) × P(B). This is the multiplication rule for dependent events.
Tree diagrams
For sequential events with conditional probabilities, tree diagrams are essential. Second branches show P(B|A) and P(B|A').
P(B) = P(B|A)P(A) + P(B|A')P(A')
This is the law of total probability — sum over all branches that lead to B.
Bayes-style problems
To find P(A|B) when given P(B|A), use a tree diagram or the formula directly:
P(A|B) = P(B|A) × P(A) / P(B)
where P(B) = P(B|A)P(A) + P(B|A')P(A')
Example1% have a disease. Test is 95% accurate for positives, 98% accurate for negatives. Given a positive test, find P(actually has disease).
P(D|+) = (0.95 × 0.01) / (0.95 × 0.01 + 0.02 × 0.99) = 0.324
Two-way tables
Organise data into rows and columns with totals. Read conditional probabilities by restricting to one row or column:
P(A|B) = count in (A ∩ B) / count in B
When the question says "given that", you MUST use conditional probability. Restricting the sample space is the key idea — your denominator changes.
For complex problems, always draw the tree diagram first. Label every branch with its probability, then identify which end-branches you need and apply the appropriate formula.
Regression & Correlation Ch 1
PMCC (r)
The product moment correlation coefficient measures the strength and direction of the linear relationship between two variables. Calculated from:
r = S_xy / √(S_xx × S_yy)

S_xx = Σx² − (Σx)²/n
S_yy = Σy² − (Σy)²/n
S_xy = Σxy − (Σx)(Σy)/n
r = 1: perfect positive linear; r = −1: perfect negative linear; r = 0: no linear correlation.
Spearman's rank (r_s)
Rank the data for each variable, then calculate:
r_s = 1 − (6Σd²) / (n(n² − 1))
where d = difference in ranks
Use when data is not linear, or for ordinal data. Values between −1 and 1. For tied ranks, assign the average rank.
Regression line y on x
y = a + bx
b = S_xy / S_xx
a = ȳ − bx̄
The regression line always passes through (x̄, ȳ). Use only to predict y from x.
Interpolation vs extrapolation
Interpolation: predicting within the range of the data — reliable.
Extrapolation: predicting beyond the range — unreliable because the linear relationship may not hold outside the observed data.
Exam phrasing"This is extrapolation as the value lies outside the range of the data, so the prediction may be unreliable."
Interpreting r
Always interpret in context: "There is a strong positive correlation between [x variable] and [y variable], suggesting that as [x] increases, [y] tends to increase."
Correlation does not imply causation.
You can only use the regression line y on x to predict y from a given x. Never use it to predict x from y — that requires the x on y regression line (which has a different equation).
For Spearman's rank with tied values: if two items share rank 3 and 4, give both rank 3.5. Then continue numbering from 5. The formula still works with averaged ranks.
Hypothesis Testing with Normal Ch 5
Testing a mean
When X ~ N(μ, σ²) with σ known, and you take a sample of size n with sample mean X̄:
X̄ ~ N(μ, σ²/n)

Test statistic: Z = (X̄ − μ₀) / (σ/√n)
where μ₀ is the value of μ under H₀.
Hypotheses
H₀: μ = μ₀
H₁: μ < μ₀ (one-tail, lower)
H₁: μ > μ₀ (one-tail, upper)
H₁: μ ≠ μ₀ (two-tail)
Critical values
Common z critical values:
5% one-tail: z = 1.6449
5% two-tail: z = ±1.9600
1% one-tail: z = 2.3263
1% two-tail: z = ±2.5758
Reject H₀ if the test statistic falls in the critical region.
Full method
1. State H₀ and H₁
2. State significance level
3. Identify distribution of X̄ under H₀: X̄ ~ N(μ₀, σ²/n)
4. Calculate test statistic z = (x̄ − μ₀)/(σ/√n)
5. Compare with critical value
6. State conclusion: reject or do not reject H₀
7. Interpret in context
Linking to Year 12
This extends the binomial hypothesis test from Year 12. Instead of testing a proportion p using binomial, you test a mean μ using the normal distribution. The logic (H₀, H₁, critical region, conclusion) is identical.
The variance of X̄ is σ²/n, NOT σ². Dividing by n reflects that sample means have less variability than individual observations. Forgetting to divide by √n when standardising is a very common error.
If the question gives σ² (variance), remember to take the square root when standardising. The denominator of the test statistic is σ/√n, not σ²/n.
