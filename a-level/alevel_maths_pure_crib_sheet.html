A-Level Pure Maths — Year 12 Crib Sheet

Edexcel · AS/Year 1 · Core Pure Topics
Algebra & Functions Ch 1–4
Laws of indices
\(a^m \times a^n = a^{m+n}\)  ·  \(a^m \div a^n = a^{m-n}\)  ·  \((a^m)^n = a^{mn}\)
\(a^0 = 1\)  ·  \(a^{-n} = \dfrac{1}{a^n}\)  ·  \(a^{\frac{1}{2}} = \sqrt{a}\)  ·  \(a^{\frac{m}{n}} = \sqrt[n]{a^m}\)
Surds
Simplify: \(\sqrt{12} = 2\sqrt{3}\). Rationalise denominators:
\(\dfrac{a}{\sqrt{b}} \to \dfrac{a\sqrt{b}}{b}\)  ·  \(\dfrac{a}{p+\sqrt{q}} \to \dfrac{a(p-\sqrt{q})}{p^2-q}\)
Completing the square
\(x^2 + bx + c = \left(x + \dfrac{b}{2}\right)^2 - \left(\dfrac{b}{2}\right)^2 + c\)
Gives vertex form: vertex at \((-b/2,\; c - b^{2}/4)\). Minimum/maximum read directly.
Discriminant
\(\Delta = b^2 - 4ac\)
\(\Delta > 0\) → two distinct real roots · \(\Delta = 0\) → one repeated root · \(\Delta < 0\) → no real roots
Quadratic formula
\(x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}\)
Simultaneous eqns
Linear + quadratic: substitute linear into quadratic. Solve resulting quadratic — may give 0, 1, or 2 solution pairs.
Example\(y = x + 1\), \(x^{2} + y^{2} = 5\) → \(x^{2} + (x+1)^{2} = 5\) → \(2x^{2} + 2x - 4 = 0\)
Inequalities
Linear: solve as equations (flip sign if \(\times / \div\) by negative).
Quadratic: solve = 0 first, sketch parabola, read off solution set.
\(x^2 - 5x + 6 < 0 \to (x-2)(x-3) < 0 \to 2 < x < 3\)
Factor theorem
If \(f(a) = 0\) then \((x - a)\) is a factor of \(f(x)\)
Use to find roots of cubics. Then algebraic division or inspection to fully factorise.
Graph transforms
\(y = f(x) + a\) → translate \((0,\, a)\) ↑
\(y = f(x + a)\) → translate \((-a,\, 0)\) ←
\(y = af(x)\) → vertical stretch, scale factor \(a\)
\(y = f(ax)\) → horizontal stretch, scale factor \(1/a\)
\(y = -f(x)\) → reflect in x-axis
\(y = f(-x)\) → reflect in y-axis
\(f(x + a)\) moves LEFT (opposite sign). \(f(ax)\) compresses by factor \(a\), not stretches. These trip up almost everyone.
Sketch cubics: find roots via factor theorem, check sign of leading coefficient for end behaviour, plot y-intercept.
Coordinate Geometry Ch 5–6
Straight line forms
\(y = mx + c\)  (slope-intercept)
\(y - y_1 = m(x - x_1)\)  (point-gradient)
Gradient
\(m = \dfrac{y_2 - y_1}{x_2 - x_1}\)
Parallel & perpendicular
Parallel: \(m_1 = m_2\)
Perpendicular: \(m_1 \times m_2 = -1\)
Distance & midpoint
\(d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}\)
Midpoint \(= \left(\dfrac{x_1+x_2}{2},\, \dfrac{y_1+y_2}{2}\right)\)
Equation of a circle
\((x - a)^2 + (y - b)^2 = r^2\)
Centre \((a, b)\), radius \(r\). Expand to get \(x^{2} + y^{2} + Dx + Ey + F = 0\) — complete the square to recover centre/radius.
Tangent / normal to circle
Radius to point of tangency is perpendicular to tangent.
1. Find gradient of radius (centre → point)
2. Tangent gradient \(= -1/m_{\text{radius}}\)
3. Use point-gradient form for tangent equation
Normal passes through centre — same gradient as radius.
To show a point lies on a circle: substitute into the equation and verify LHS = RHS. To find intersections with a line: substitute and solve the resulting quadratic.
Proof Ch 7
By deduction
Start from known facts/axioms, use logical steps to reach the conclusion. Most common proof type at AS level.
ExampleProve \((2n+1)^{2}\) is always odd: \(= 4n^{2}+4n+1 = 2(2n^{2}+2n)+1\), which is odd ✓
By exhaustion
Check every possible case. Only works when the number of cases is finite and manageable.
Disproof by counter-example
To disprove "for all x, P(x)" — find ONE x where P(x) is false.
Example"\(n^{2} + n + 1\) is always prime" → \(n = 4\): \(16+4+1 = 21 = 3 \times 7\). Disproved ✓
Use \(2n\) for even, \(2n+1\) for odd, \(n^{2}\) for square numbers. These representations are your proof toolkit.
Differentiation Ch 12–13
First principles
\(f'(x) = \lim_{h \to 0} \dfrac{f(x+h) - f(x)}{h}\)
The formal definition — you must be able to derive simple results (e.g. \(x^{n}\) for small \(n\)) from this.
Power rule
\(y = ax^n \;\to\; \dfrac{dy}{dx} = anx^{n-1}\)
Works for fractional and negative \(n\). Rewrite roots/fractions first:
\(\sqrt{x} = x^{\frac{1}{2}} \;\to\; \dfrac{dy}{dx} = \tfrac{1}{2}x^{-\frac{1}{2}}\)
\(\dfrac{1}{x^2} = x^{-2} \;\to\; \dfrac{dy}{dx} = -2x^{-3}\)
Gradient at a point
Substitute x-value into \(\frac{dy}{dx}\). This gives the gradient of the tangent to the curve at that point.
Tangent & normal
At point (a, f(a)):
Tangent: \(y - f(a) = f'(a)(x - a)\)
Normal: \(y - f(a) = -\dfrac{1}{f'(a)}(x - a)\)
Stationary points
Set \(\dfrac{dy}{dx} = 0\), solve for \(x\)
Classify using second derivative:
\(\dfrac{d^2y}{dx^2} > 0\) → minimum
\(\dfrac{d^2y}{dx^2} < 0\) → maximum
\(\dfrac{d^2y}{dx^2} = 0\) → test further (could be point of inflection)
Increasing / decreasing
\(f'(x) > 0\) → \(f\) is increasing
\(f'(x) < 0\) → \(f\) is decreasing
Find where \(\frac{dy}{dx}\) changes sign to identify intervals.
When \(\frac{d^{2}y}{dx^{2}} = 0\), you CANNOT conclude "point of inflection" — you must check sign changes in \(\frac{dy}{dx}\) either side. Examiners penalise this.
Integration Ch 14
Reverse of differentiation
\(\displaystyle\int x^n\, dx = \frac{x^{n+1}}{n+1} + c \quad (n \neq -1)\)
Works for fractional/negative powers. Integrate term by term.
Finding f(x) from f′(x)
Integrate, then use a known point to find c.
Example\(f'(x) = 3x^{2}\), \(f(1) = 5\) → \(f(x) = x^{3} + c\) → \(1 + c = 5\) → \(c = 4\) → \(f(x) = x^{3} + 4\)
Definite integrals
\(\displaystyle\int_a^b f(x)\, dx = \Big[F(x)\Big]_a^b = F(b) - F(a)\)
No \(+c\) needed. Result gives the signed area under the curve between \(x = a\) and \(x = b\).
Area under a curve
Area between curve \(y = f(x)\), x-axis, \(x = a\) and \(x = b\). If curve dips below x-axis, that region gives a negative integral — take absolute value and add separately.
Area between curves
\(\displaystyle\text{Area} = \int_a^b \big[f(x) - g(x)\big]\, dx\)
Where \(f(x)\) is the upper curve. Find intersection points first (set \(f = g\)).
ALWAYS add \(+c\) for indefinite integrals. Losing the constant of integration loses marks every single time.
If the area comes out negative, check: is the curve below the x-axis in that interval? Split the integral at the roots.
Exponentials & Logarithms Ch 14
eˣ and ln(x)
\(e^{x}\) and \(\ln(x)\) are inverse functions. \(e \approx 2.71828\ldots\)
\(\ln(e^x) = x\)  ·  \(e^{\ln x} = x\)  (\(x > 0\))
The gradient of \(y = e^{x}\) is \(e^{x}\) (it differentiates to itself).
Laws of logarithms
\(\log(ab) = \log a + \log b\)
\(\log(a/b) = \log a - \log b\)
\(\log(a^n) = n\log a\)
\(\log(1) = 0\)  ·  \(\log_a(a) = 1\)
These work for any base — \(\log\), \(\ln\), \(\log_{10}\), etc.
Solving aˣ = b
Take logs of both sides:
\(a^x = b \;\to\; x\ln a = \ln b \;\to\; x = \dfrac{\ln b}{\ln a}\)
Change of base
\(\log_a(b) = \dfrac{\ln b}{\ln a} = \dfrac{\log b}{\log a}\)
Growth & decay models
\(N = N_0 e^{kt}\)
\(k > 0\) → growth · \(k < 0\) → decay. \(N_{0}\) = initial value, \(t\) = time. To find \(t\) for a given \(N\): rearrange and use \(\ln\).
\(\ln(a + b) \neq \ln a + \ln b\). The log laws only work for products, quotients, and powers — never sums.
Trigonometry Ch 9–10
Exact values
$$\begin{array}{c|ccccc} & 0° & 30° & 45° & 60° & 90° \\ \hline \sin & 0 & \tfrac{1}{2} & \tfrac{\sqrt{2}}{2} & \tfrac{\sqrt{3}}{2} & 1 \\ \cos & 1 & \tfrac{\sqrt{3}}{2} & \tfrac{\sqrt{2}}{2} & \tfrac{1}{2} & 0 \\ \tan & 0 & \tfrac{\sqrt{3}}{3} & 1 & \sqrt{3} & — \end{array}$$
Sine & cosine rules
Sine rule: \(\dfrac{a}{\sin A} = \dfrac{b}{\sin B} = \dfrac{c}{\sin C}\)
Cosine rule: \(a^2 = b^2 + c^2 - 2bc\cos A\)
Area \(= \tfrac{1}{2}ab\sin C\)
Sine rule: need angle-side opposite pair. Cosine rule: need SAS or SSS.
Identities
\(\sin^2\theta + \cos^2\theta \equiv 1\)
\(\tan\theta \equiv \dfrac{\sin\theta}{\cos\theta}\)
Use to simplify expressions and solve equations. Rearrange: \(\sin^{2}\theta = 1 - \cos^{2}\theta\), etc.
Solving trig equations
1. Reduce to \(\sin\theta / \cos\theta / \tan\theta = k\) using identities
2. Find principal value from calculator
3. Use CAST diagram or symmetry for all solutions in range
\(\sin\theta = \sin\alpha \;\to\; \theta = \alpha\) or \(\theta = 180° - \alpha\) (+360°n)
\(\cos\theta = \cos\alpha \;\to\; \theta = \alpha\) or \(\theta = 360° - \alpha\) (+360°n)
\(\tan\theta = \tan\alpha \;\to\; \theta = \alpha\) (+180°n)
Radians
\(\pi \text{ rad} = 180°\)  ·  \(1 \text{ rad} \approx 57.3°\)
Arc length: \(s = r\theta\)
Sector area: \(A = \tfrac{1}{2}r^2\theta\)
Segment area: \(A = \tfrac{1}{2}r^2(\theta - \sin\theta)\)
\(\theta\) must be in radians for these formulas.
Make sure your calculator is in the correct mode (degrees or radians). This catches students in every exam sitting.
For trig equations: always write down the full general solution, then list values in the given range. Show the CAST diagram if asked to "show working".
