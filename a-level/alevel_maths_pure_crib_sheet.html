A-Level Pure Maths — Year 12 Crib Sheet

Edexcel · AS/Year 1 · Core Pure Topics
Algebra & Functions Ch 1–4
Laws of indices
aᵐ × aⁿ = aᵐ⁺ⁿ  ·  aᵐ ÷ aⁿ = aᵐ⁻ⁿ  ·  (aᵐ)ⁿ = aᵐⁿ
a⁰ = 1  ·  a⁻ⁿ = 1/aⁿ  ·  a^(½) = √a  ·  a^(m/n) = ⁿ√(aᵐ)
Surds
Simplify: √12 = 2√3. Rationalise denominators:
a/√b → a√b/b  ·  a/(p+√q) → a(p−√q)/(p²−q)
Completing the square
x² + bx + c = (x + b/2)² − (b/2)² + c
Gives vertex form: vertex at (−b/2, c − b²/4). Minimum/maximum read directly.
Discriminant
Δ = b² − 4ac
Δ > 0 → two distinct real roots · Δ = 0 → one repeated root · Δ < 0 → no real roots
Quadratic formula
x = (−b ± √(b²−4ac)) / 2a
Simultaneous eqns
Linear + quadratic: substitute linear into quadratic. Solve resulting quadratic — may give 0, 1, or 2 solution pairs.
Exampley = x + 1, x² + y² = 5 → x² + (x+1)² = 5 → 2x² + 2x − 4 = 0
Inequalities
Linear: solve as equations (flip sign if ×/÷ by negative).
Quadratic: solve = 0 first, sketch parabola, read off solution set.
x² − 5x + 6 < 0 → (x−2)(x−3) < 0 → 2 < x < 3
Factor theorem
If f(a) = 0 then (x − a) is a factor of f(x)
Use to find roots of cubics. Then algebraic division or inspection to fully factorise.
Graph transforms
y = f(x) + a → translate (0, a) ↑
y = f(x + a) → translate (−a, 0) ←
y = af(x) → vertical stretch, scale factor a
y = f(ax) → horizontal stretch, scale factor 1/a
y = −f(x) → reflect in x-axis
y = f(−x) → reflect in y-axis
f(x + a) moves LEFT (opposite sign). f(ax) compresses by factor a, not stretches. These trip up almost everyone.
Sketch cubics: find roots via factor theorem, check sign of leading coefficient for end behaviour, plot y-intercept.
Coordinate Geometry Ch 5–6
Straight line forms
y = mx + c  (slope-intercept)
y − y₁ = m(x − x₁)  (point-gradient)
Gradient
m = (y₂ − y₁) / (x₂ − x₁)
Parallel & perpendicular
Parallel: m₁ = m₂
Perpendicular: m₁ × m₂ = −1
Distance & midpoint
d = √((x₂−x₁)² + (y₂−y₁)²)
Midpoint = ((x₁+x₂)/2, (y₁+y₂)/2)
Equation of a circle
(x − a)² + (y − b)² = r²
Centre (a, b), radius r. Expand to get x² + y² + Dx + Ey + F = 0 — complete the square to recover centre/radius.
Tangent / normal to circle
Radius to point of tangency is perpendicular to tangent.
1. Find gradient of radius (centre → point)
2. Tangent gradient = −1/m_radius
3. Use point-gradient form for tangent equation
Normal passes through centre — same gradient as radius.
To show a point lies on a circle: substitute into the equation and verify LHS = RHS. To find intersections with a line: substitute and solve the resulting quadratic.
Proof Ch 7
By deduction
Start from known facts/axioms, use logical steps to reach the conclusion. Most common proof type at AS level.
ExampleProve (2n+1)² is always odd: = 4n²+4n+1 = 2(2n²+2n)+1, which is odd ✓
By exhaustion
Check every possible case. Only works when the number of cases is finite and manageable.
Disproof by counter-example
To disprove "for all x, P(x)" — find ONE x where P(x) is false.
Example"n² + n + 1 is always prime" → n = 4: 16+4+1 = 21 = 3×7. Disproved ✓
Use 2n for even, 2n+1 for odd, n² for square numbers. These representations are your proof toolkit.
Differentiation Ch 12–13
First principles
f′(x) = lim(h→0) [f(x+h) − f(x)] / h
The formal definition — you must be able to derive simple results (e.g. xⁿ for small n) from this.
Power rule
y = axⁿ → dy/dx = anxⁿ⁻¹
Works for fractional and negative n. Rewrite roots/fractions first:
√x = x^(½) → dy/dx = ½x^(−½)
1/x² = x⁻² → dy/dx = −2x⁻³
Gradient at a point
Substitute x-value into dy/dx. This gives the gradient of the tangent to the curve at that point.
Tangent & normal
At point (a, f(a)):
Tangent: y − f(a) = f′(a)(x − a)
Normal: y − f(a) = −1/f′(a) · (x − a)
Stationary points
Set dy/dx = 0, solve for x
Classify using second derivative:
d²y/dx² > 0 → minimum
d²y/dx² < 0 → maximum
d²y/dx² = 0 → test further (could be point of inflection)
Increasing / decreasing
f′(x) > 0 → f is increasing
f′(x) < 0 → f is decreasing
Find where dy/dx changes sign to identify intervals.
When d²y/dx² = 0, you CANNOT conclude "point of inflection" — you must check sign changes in dy/dx either side. Examiners penalise this.
Integration Ch 14
Reverse of differentiation
∫xⁿ dx = xⁿ⁺¹/(n+1) + c   (n ≠ −1)
Works for fractional/negative powers. Integrate term by term.
Finding f(x) from f′(x)
Integrate, then use a known point to find c.
Examplef′(x) = 3x², f(1) = 5 → f(x) = x³ + c → 1 + c = 5 → c = 4 → f(x) = x³ + 4
Definite integrals
∫ₐᵇ f(x) dx = [F(x)]ₐᵇ = F(b) − F(a)
No +c needed. Result gives the signed area under the curve between x = a and x = b.
Area under a curve
Area between curve y = f(x), x-axis, x = a and x = b. If curve dips below x-axis, that region gives a negative integral — take |value| and add separately.
Area between curves
Area = ∫ₐᵇ [f(x) − g(x)] dx
Where f(x) is the upper curve. Find intersection points first (set f = g).
ALWAYS add +c for indefinite integrals. Losing the constant of integration loses marks every single time.
If the area comes out negative, check: is the curve below the x-axis in that interval? Split the integral at the roots.
Exponentials & Logarithms Ch 14
eˣ and ln(x)
eˣ and ln(x) are inverse functions. e ≈ 2.71828...
ln(eˣ) = x  ·  e^(ln x) = x  (x > 0)
The gradient of y = eˣ is eˣ (it differentiates to itself).
Laws of logarithms
log(ab) = log a + log b
log(a/b) = log a − log b
log(aⁿ) = n log a
log(1) = 0  ·  logₐ(a) = 1
These work for any base — log, ln, log₁₀, etc.
Solving aˣ = b
Take logs of both sides:
aˣ = b → x ln a = ln b → x = ln b / ln a
Change of base
logₐ(b) = ln b / ln a = log b / log a
Growth & decay models
N = N₀ eᵏᵗ
k > 0 → growth · k < 0 → decay. N₀ = initial value, t = time. To find t for a given N: rearrange and use ln.
ln(a + b) ≠ ln a + ln b. The log laws only work for products, quotients, and powers — never sums.
Trigonometry Ch 9–10
Exact values
0° 30° 45° 60° 90° sin 0 1/2 √2/2 √3/2 1 cos 1 √3/2 √2/2 1/2 0 tan 0 √3/3 1 √3 —
Sine & cosine rules
Sine rule: a/sinA = b/sinB = c/sinC
Cosine rule: a² = b² + c² − 2bc cosA
Area = ½ab sinC
Sine rule: need angle-side opposite pair. Cosine rule: need SAS or SSS.
Identities
sin²θ + cos²θ ≡ 1
tanθ ≡ sinθ / cosθ
Use to simplify expressions and solve equations. Rearrange: sin²θ = 1 − cos²θ, etc.
Solving trig equations
1. Reduce to sin/cos/tanθ = k using identities
2. Find principal value from calculator
3. Use CAST diagram or symmetry for all solutions in range
sinθ = sinα → θ = α or θ = 180°−α (+ 360°n)
cosθ = cosα → θ = α or θ = 360°−α (+ 360°n)
tanθ = tanα → θ = α (+ 180°n)
Radians
π rad = 180°  ·  1 rad ≈ 57.3°
Arc length: s = rθ
Sector area: A = ½r²θ
Segment area: A = ½r²(θ − sinθ)
θ must be in radians for these formulas.
Make sure your calculator is in the correct mode (degrees or radians). This catches students in every exam sitting.
For trig equations: always write down the full general solution, then list values in the given range. Show the CAST diagram if asked to "show working".
