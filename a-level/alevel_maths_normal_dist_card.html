Using the Table

P(Z < z)
Read directly from the table
P(Z > z)
= 1 − P(Z < z)
P(Z < −z)
= 1 − P(Z < z)  (symmetry)
P(a < Z < b)
= P(Z < b) − P(Z < a)

Inverse Normal — Going Backwards

Given a probability, find the value
If P(X < a) = p, then:
1. Find z where P(Z < z) = p   (from tables, reading backwards)
2. Then a = μ + zσ
Worked Example: X ~ N(50, 16). Find a such that P(X < a) = 0.9
1
Identify: μ = 50, σ² = 16, so σ = 4
2
Find z where P(Z < z) = 0.9 from tables z = 1.2816
3
Convert back: a = μ + zσ a = 50 + 1.2816 × 4 = 55.13 (2 d.p.)
a = 55.13

Finding μ and σ — Simultaneous Equations

The Method
Given two probability statements about X ~ N(μ, σ²):
P(X < a) = p1  ⇒  z1 from tables  ⇒  a = μ + z1σ  … [1]
P(X < b) = p2  ⇒  z2 from tables  ⇒  b = μ + z2σ  … [2]
Subtract one equation from the other to eliminate μ and find σ, then substitute back to find μ.
Worked Example: P(X < 62) = 0.9 and P(X < 38) = 0.1. Find μ and σ.
1
From tables: P(Z < z) = 0.9 gives z1 = 1.2816 62 = μ + 1.2816σ … [1]
2
P(Z < z) = 0.1 ⇒ by symmetry z2 = −1.2816 38 = μ − 1.2816σ … [2]
3
[1] − [2]: 62 − 38 = 2(1.2816)σ 24 = 2.5632σ ⇒ σ = 9.365 (3 d.p.)
4
Substitute into [1]: μ = 62 − 1.2816 × 9.365 μ = 50
μ = 50, σ = 9.37 (3 s.f.)

Normal Approximation to Binomial

When to use it
n is large
np > 5
n(1 − p) > 5
The approximation
X ~ B(n, p)
≈ Y ~ N(np, np(1−p))
Mean = np, Variance = np(1−p)
Continuity Correction — This is crucial!
Binomial is discrete (whole numbers), Normal is continuous. You must adjust by ±0.5 to bridge the gap.
Binomial (discrete) Normal (continuous)
P(X ≤ n) P(Y < n + 0.5)
P(X < n) P(Y < n − 0.5)
P(X ≥ n) P(Y > n − 0.5)
P(X > n) P(Y > n + 0.5)
P(X = n) P(n − 0.5 < Y < n + 0.5)

Common Mistakes

Using σ² instead of σ (or vice versa) in the standardising formula — z = (x − μ) / σ, not σ²
Forgetting the continuity correction when approximating binomial with normal
Getting the inequality direction wrong after standardising — always redraw the bell curve
Not drawing a sketch — always shade the area you want on a bell curve first
