Integration by Substitution
"Spot the inner function whose derivative is also there"
1 Choose u (usually the inner function)
2 Find \(\dfrac{du}{dx}\), rearrange to get \(dx = \ldots\,du\)
3 Replace everything in terms of u (no x left!)
4 Integrate in terms of u
5 Substitute back (or change limits if definite)

Example 1: \(\int 2x \cdot e^{x^2}\,dx\)

Let \(u = x^2, \quad \dfrac{du}{dx} = 2x, \quad \text{so } du = 2x\,dx\)
\(\int e^u\,du = e^u + c\)
\(= e^{x^2} + c\) ✓

Example 2: \(\int 3x^2(x^3 + 1)^4\,dx\)

Let \(u = x^3 + 1, \quad \dfrac{du}{dx} = 3x^2, \quad \text{so } du = 3x^2\,dx\)
\(\int u^4\,du = \dfrac{u^5}{5} + c\)
\(= \dfrac{(x^3 + 1)^5}{5} + c\) ✓
Integration by Parts
\(\int u\,dv = uv - \int v\,du\)
"LIATE: Logs, Inverse trig, Algebraic, Trig, Exponentials"
Choose u from the left of LIATE (it simplifies when differentiated).
Choose dv from the right (it stays manageable when integrated).
1 Label u and dv using LIATE
2 Find du (differentiate u) and v (integrate dv)
3 Apply the formula: \(uv - \int v\,du\)
4 Integrate the remaining integral

Example 1: \(\int x \cdot e^x\,dx\)

\(u = x, \quad dv = e^x\,dx\)
\(du = dx, \quad v = e^x\)
\(= x \cdot e^x - \int e^x\,dx\)
\(= xe^x - e^x + c = e^x(x - 1) + c\) ✓

Example 2: \(\int x^2 \sin(x)\,dx\) — needs parts TWICE

\(u = x^2,\; dv = \sin(x)\,dx \rightarrow du = 2x\,dx,\; v = -\cos(x)\)
\(= -x^2\cos(x) + \int 2x\cos(x)\,dx\)
Now parts again: \(u = 2x,\; dv = \cos(x)\,dx\)
\(= -x^2\cos(x) + 2x\sin(x) - \int 2\sin(x)\,dx\)
\(= -x^2\cos(x) + 2x\sin(x) + 2\cos(x) + c\) ✓
Partial Fractions
"Split the fraction, integrate each bit to ln"
When the denominator factors into linear terms, split into simpler fractions that each integrate to ln form.
1 Factor the denominator
2 Split: \(\dfrac{A}{x-a} + \dfrac{B}{x-b}\)
3 Find A and B (cover-up method or equate coefficients)
4 Integrate each term: \(A\ln|x-a| + B\ln|x-b| + c\)

Example: \(\int \dfrac{3}{(x-1)(x+2)}\,dx\)

\(\dfrac{3}{(x-1)(x+2)} = \dfrac{A}{x-1} + \dfrac{B}{x+2}\)
Cover-up: \(x=1 \rightarrow A = \frac{3}{3} = 1, \quad x=-2 \rightarrow B = \frac{3}{-3} = -1\)
\(\int \left[\dfrac{1}{x-1} - \dfrac{1}{x+2}\right] dx\)
\(= \ln|x-1| - \ln|x+2| + c\) ✓
Definite Integrals & Areas
1 Area under curve between a and b: \(\int_a^b f(x)\,dx\)
2 Area between two curves: \(\int_a^b [f(x) - g(x)]\,dx\)
3 If curve goes below x-axis: the integral is negative — take the absolute value for area
4 Trapezium rule (numerical approximation):
\(\dfrac{h}{2}\left[y_0 + 2(y_1 + y_2 + \ldots + y_{n-1}) + y_n\right]\)
where \(h = \dfrac{b - a}{n}\) is the strip width. More strips = better approximation.

Common Mistakes

  • Forgetting +c on indefinite integrals. Every. Single. Time. Free mark lost.
  • Forgetting the 1/k factor:
    \(\int \sin(3x)\,dx\) = \(-\cos(3x) + c\) → \(-\frac{1}{3}\cos(3x) + c\)
  • Not changing limits when using substitution on a definite integral: if you substitute u, the limits must change from x-values to u-values. Otherwise substitute back to x before evaluating.
  • Choosing the wrong u in integration by parts: use LIATE. If \(u = e^x\) and \(dv = x\,dx\), you've made it harder, not easier.
