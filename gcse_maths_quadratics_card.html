Quadratics & Completing the Square

Three methods, one goal: solve ax² + bx + c = 0

1. What is a Quadratic?

Any equation where x² is the highest power

ax² + bx + c = 0
  • a = coefficient of x² (can't be 0)
  • b = coefficient of x
  • c = constant term
  • The graph is always a parabola
  • If a > 0 it's a U shape — if a < 0 it's an ∩ shape

2. Three Ways to Solve ax² + bx + c = 0

Try first

Factorising

When the numbers are "nice" (integer solutions)
  • Find two numbers that multiply to give c
  • …and add to give b
  • Write as two brackets
  • Set each bracket = 0
Worked Example
x² + 5x + 6 = 0
What multiplies to 6 and adds to 5?
2 × 3 = 6    2 + 3 = 5   ✔
(x + 2)(x + 3) = 0
x = −2 or x = −3
Always works

Quadratic Formula

When it won't factorise — gives exact or decimal answers
x = (−b ± √(b²−4ac)) / 2a
  • Identify a, b, and c
  • Substitute into the formula
  • The ± gives two answers
Example: 2x² + 3x − 2 = 0
a=2, b=3, c=−2
x = (−3 ± √(9+16)) / 4
x = (−3 ± √25) / 4
x = (−3 ± 5) / 4
x = 0.5 or x = −2
Higher tier

Completing the Square

When you need the turning point or exact surd form
x² + bx + c → (x + b/2)² − (b/2)² + c
  • Halve the coefficient of x
  • Write (x + half)²
  • Subtract (half)²
  • Add c back on
Example: x² + 6x + 2
Half of 6 = 3
(x + 3)² − 9 + 2
(x + 3)² − 7
Turning point: (−3, −7)

The Discriminant: b² − 4ac tells you how many solutions

b²−4ac > 0
2 Solutions
Parabola crosses x-axis twice
b²−4ac = 0
1 Solution
Parabola just touches x-axis (repeated root)
b²−4ac < 0
No Real Solutions
Parabola doesn't reach x-axis

3. Which Method Should I Use?

The Decision Guide

Does it factorise easily?
→
Factorise
Says "give to 2 d.p." or "give in surd form"?
→
Quadratic Formula
Asks for the turning point?
→
Complete the Square
Not sure? The formula ALWAYS works. When in doubt, use the formula.

4. Sketching Quadratic Graphs

Four things to find, then sketch

1
Find the roots (where y = 0) Use any of the three solving methods
2
Find the y-intercept (set x = 0) It's just c — the constant term
3
Find the turning point x = −b/2a, then substitute back for y
4
U or ∩? Check the sign of a a > 0 = U shape  |  a < 0 = ∩ shape

5. Common Mistakes

Don't lose marks on these

  • Forgetting the ± in the quadratic formula — there are two solutions!
  • Writing x = −2, 3 instead of x = −2 or x = 3
  • Calculating b² wrong: it's b × b, not b × 2
  • Sign errors when completing the square — watch the minus!
  • Forgetting to set the equation = 0 before solving
