Geometry & Trigonometry
Angles on parallel lines & in triangles
Alternate angles (Z-shape) are equal. Corresponding angles (F-shape) are equal. Co-interior angles (C/U-shape) sum to 180°. Angles in a triangle sum to 180°.
Exterior angle of triangle = sum of two opposite interior angles
Always mark angles on the diagram. State the reason for each step, e.g. "alternate angles are equal".
120, 183
Angles in polygons
Interior angle sum = (n - 2) x 180°
Each interior angle (regular) = (n - 2) x 180° / n
Each exterior angle (regular) = 360° / n
Interior + exterior = 180° at each vertex. To find n from an exterior angle: n = 360 / exterior angle.
ExampleRegular octagon: exterior = 360/8 = 45°, interior = 135°
If asked "how many sides?" and given an interior angle, find the exterior first (180 - interior), then divide 360 by it.
123
Circle theorems
Key theorems: (1) Angle at centre = 2 x angle at circumference. (2) Angle in semicircle = 90°. (3) Angles in same segment are equal. (4) Opposite angles of cyclic quadrilateral sum to 180°. (5) Tangent is perpendicular to radius. (6) Two tangents from external point are equal length. (7) Alternate segment theorem: angle between tangent and chord = angle in alternate segment.
You MUST state the theorem name or a clear description of the rule used. Just writing the angle value scores 0.
Draw in extra lines (radii, diameters) to reveal isosceles triangles inside the circle.
183
Alternate segment theorem
The angle between a tangent to a circle and a chord drawn from the point of tangency equals the inscribed angle subtended by the chord on the opposite side.
How to spot itLook for a tangent touching the circle at a point where a chord also starts. The angle between them equals the angle at the far end of the chord in the other segment.

Area of sectors
Sector area = (θ / 360) x π r²
Arc length = (θ / 360) x 2πr
Where θ is the angle at the centre in degrees, r is the radius.
If given arc length and asked for area, find θ first from the arc formula, then use it in the area formula.

Trigonometry: finding angles
In a right-angled triangle, use SOHCAHTOA to find unknown angles:
θ = sin⁻¹(O/H) or θ = cos⁻¹(A/H) or θ = tan⁻¹(O/A)
Label the sides relative to the angle: Opposite, Adjacent, Hypotenuse. Choose the ratio that uses the two known sides.
Make sure your calculator is in DEGREES mode, not radians.
150
Area rule (non-right triangles)
Area = ½ ab sin(C)
Use when you know two sides and the included angle (the angle between those two sides). This is on the formula sheet but practise using it.
Examplea = 8cm, b = 5cm, C = 30°: Area = ½ x 8 x 5 x sin(30°) = ½ x 8 x 5 x 0.5 = 10 cm²
203
Area of triangles & trapeziums
Triangle: Area = ½ x base x height
Trapezium: Area = ½(a + b) x h
Where a and b are the parallel sides, h is the perpendicular height (not the slant height).
The height must be perpendicular to the base. If the triangle is not right-angled, you may need to draw a construction line for the height.

Algebra & Functions
Completing the square
Rewrite ax² + bx + c in the form a(x + p)² + q.
x² + bx + c = (x + b/2)² - (b/2)² + c
Examplex² + 6x + 2 = (x + 3)² - 9 + 2 = (x + 3)² - 7
The turning point is at (-p, q). If coefficient of x² is not 1, factor it out first.
Sign trap: (x + 3)² means the turning point x-value is -3, not +3.
209
Turning point from completed square
Once in the form a(x + p)² + q, the vertex (turning point) is at (-p, q). If a > 0, it is a minimum. If a < 0, it is a maximum.
y = a(x + p)² + q → minimum/maximum at (-p, q)
This method is more reliable than using calculus for quadratics at GCSE. The completed square also tells you the line of symmetry: x = -p.
209, 160
Factorising quadratics (ax² + bx + c = 0)
When a = 1: find two numbers that multiply to c and add to b. When a ≠ 1: multiply a x c, find two numbers that multiply to ac and add to b, then split the middle term and factorise in pairs.
6x² + x - 2 = 0 → ac = -12, factors: 4 and -3 6x² + 4x - 3x - 2 = 2x(3x+2) - 1(3x+2) = (2x-1)(3x+2)
Always check by expanding your answer back out. If the quadratic doesn't factorise neatly, use the quadratic formula instead.

Constructing & solving equations
Read the problem carefully. Define a variable (let x = ...). Write expressions for each quantity, form an equation, then solve.
Example"Three consecutive numbers add to 78" → x + (x+1) + (x+2) = 78 → 3x + 3 = 78 → x = 25
Check your answer makes sense in context. If asked for a length, it must be positive.
135
Simultaneous equations
Elimination: make coefficients of one variable equal, then add/subtract. Substitution: rearrange one equation for a variable and substitute into the other.
2x + 3y = 12 ... (1) 5x - 3y = 9 ... (2) Add: 7x = 21 → x = 3, then y = 2
If one equation is linear and one quadratic, you must use substitution. Expect two pairs of answers.
162
Composite functions
fg(x) means "apply g first, then f to the result". Replace x in f with the whole of g(x).
If f(x) = 2x + 1, g(x) = x² fg(x) = f(g(x)) = 2(x²) + 1 = 2x² + 1 gf(x) = g(f(x)) = (2x + 1)² = 4x² + 4x + 1
fg(x) ≠ gf(x) in general. Order matters. Read carefully which way round it is.
215
Inverse functions
To find f⁻¹(x): write y = f(x), swap x and y, then rearrange for y.
f(x) = 3x - 5 → y = 3x - 5 Swap: x = 3y - 5 → 3y = x + 5 → y = (x + 5)/3 So f⁻¹(x) = (x + 5)/3
The graph of f⁻¹(x) is a reflection of f(x) in the line y = x. Also, ff⁻¹(x) = x always.
214
Index rules (negative indices)
a⁻ⁿ = 1/aⁿ e.g. 2⁻³ = 1/8 (a/b)⁻ⁿ = (b/a)ⁿ e.g. (2/3)⁻² = (3/2)² = 9/4
Key rules: aᵐ x aⁿ = aᵐ⁺ⁿ, aᵐ / aⁿ = aᵐ⁻ⁿ, (aᵐ)ⁿ = aᵐⁿ, a⁰ = 1.
Standard form with negative powers: 3.2 x 10⁻⁴ = 0.00032. The negative power tells you how many places to move the decimal left.

Number & Proportion
Percentage change (calculator)
% change = ((new - original) / original) x 100
Multiplier method: increase of 15% → x 1.15, decrease of 8% → x 0.92. For compound/repeated change, use powers:
After n years at r% increase: original x (1 + r/100)ⁿ
To find the original after a % change: divide by the multiplier, not subtract the percentage.

Ratio: sharing amounts
Add parts of ratio. Divide total by sum of parts to find one part. Multiply each ratio number by the value of one part.
ExampleShare £200 in ratio 3:5 → total parts = 8, one part = £25 → shares are £75 and £125
Write the units after each calculation to keep track. Check your shares add to the original total.

Direct proportion
If y is directly proportional to x: y = kx. Find k using a known pair, then use the equation.
y ∝ x → y = kx y ∝ x² → y = kx² y ∝ √x → y = k√x
Exampley ∝ x², y = 50 when x = 5 → 50 = k(25) → k = 2 → y = 2x²
Read carefully whether it's proportional to x, x², x³, or √x. The type of proportion changes the equation.

Unit conversions
Length: 1 km = 1000 m, 1 m = 100 cm, 1 cm = 10 mm. Mass: 1 kg = 1000 g, 1 tonne = 1000 kg. Capacity: 1 litre = 1000 ml, 1 cm³ = 1 ml.
Area: multiply conversion factor squared (1 m² = 10000 cm²) Volume: multiply conversion factor cubed (1 m³ = 1000000 cm³)
Don't forget to square or cube the conversion factor for area and volume conversions.
199
Bounds & error intervals
A measurement rounded to a given degree of accuracy has upper and lower bounds.
Value rounded to nearest 10: bounds = value ± 5 e.g. 340 (nearest 10): LB = 335, UB = 345
For calculations: to maximise a division, use max numerator and min denominator. To maximise a product, use max of both.
Max of a/b = UB(a) / LB(b) Min of a/b = LB(a) / UB(b)
Upper bound is never included. Write as inequality: 335 ≤ x < 345, not 335 ≤ x ≤ 345.

Surds & rationalising
√(ab) = √a x √b e.g. √12 = √4 x √3 = 2√3 √(a/b) = √a / √b
To rationalise a denominator with two terms, multiply top and bottom by the conjugate:
1/(a + √b): multiply by (a - √b)/(a - √b) Uses difference of two squares: (a+√b)(a-√b) = a² - b
Example3/(2+√5) = 3(2-√5)/((2+√5)(2-√5)) = 3(2-√5)/(4-5) = 3(2-√5)/(-1) = -6+3√5

Calculator & sig figs
Use your calculator efficiently: brackets for complex fractions, ANS key for multi-step. Round to significant figures by counting from the first non-zero digit.
0.004562 to 2 s.f. = 0.0046 (4 and 5 are the first two sig figs) 34571 to 3 s.f. = 34600

Graphs
Translating graphs
y = f(x) + a → shift UP by a y = f(x) - a → shift DOWN by a y = f(x + a) → shift LEFT by a y = f(x - a) → shift RIGHT by a
The horizontal shift is counter-intuitive: f(x + 2) moves LEFT, f(x - 2) moves RIGHT. Think: "what value of x gives the same output as f(0)?"
If asked to sketch a transformation, draw the original first, then move key points (intercepts, turning points) by the translation vector.

Estimating gradients of curves
Draw a tangent line to the curve at the given point. Choose two points on the tangent line that are far apart. Calculate gradient = rise/run.
Gradient of tangent at point P = Δy / Δx (using two points on the tangent line)
Make your tangent line long and use points where it crosses grid lines for accuracy. The examiner allows a range of acceptable answers.
A tangent just touches the curve at the point. A chord cuts through it. Don't confuse the two.
216
Quadratic graphs & turning points
y = ax² + bx + c forms a parabola. Complete the square to find the turning point without plotting.
y = (x + p)² + q → turning point at (-p, q) y = -(x + p)² + q → inverted, max at (-p, q)
The line of symmetry passes through the turning point: x = -p. Roots (if any) are where the parabola crosses the x-axis. 209, 160
Speed, Distance & Time
Calculating with speed
Speed = Distance / Time Distance = Speed x Time Time = Distance / Speed
Use consistent units throughout (e.g. km and hours for km/h, m and seconds for m/s).
km/h to m/s: divide by 3.6 (or x 1000 / 3600) m/s to km/h: multiply by 3.6
Draw the DST triangle: D on top, S and T on the bottom. Cover what you want to find.
142
Distance-time graphs
The gradient of a distance-time graph gives speed. A steeper line = faster speed. A horizontal line = stationary. A curved line = changing speed (acceleration/deceleration).
Speed = gradient = change in distance / change in time
Don't confuse distance-time with velocity-time graphs. In distance-time graphs, area under the graph has no useful meaning.
143
Velocity-time graphs
Gradient = acceleration. Area under graph = distance travelled. For a trapezium-shaped graph:
Acceleration = change in velocity / time taken Distance = area under graph = ½(a + b) x h
Compound areasSplit complex shapes into rectangles and triangles. Calculate each area separately and add them together.
For curved velocity-time graphs, estimate the area by counting squares or using the trapezium rule.
142
Bounds with speed
When distance and time are both rounded, use bounds to find max/min speed:
Max speed = UB(distance) / LB(time) Min speed = LB(distance) / UB(time)
ExampleDistance = 100m (nearest m), Time = 12s (nearest s): Max speed = 100.5/11.5 = 8.74 m/s, Min speed = 99.5/12.5 = 7.96 m/s

Statistics & Probability
Interpreting pie charts
Each sector angle represents a proportion of the whole. To find frequency: (sector angle / 360) x total. To find angle: (frequency / total) x 360.
Frequency = (angle / 360) x total Angle = (frequency / total) x 360
Check all your angles add to 360° and all frequencies add to the total. Use a protractor carefully when drawing.
128, 65
Interpreting bar charts
Read values carefully from the scale. For comparative bar charts, compare heights of corresponding bars. Note whether the y-axis starts at zero (if not, differences can be visually misleading).
If the y-axis doesn't start at 0, the visual difference between bars is exaggerated. Always read the actual values.

Example20 people total, 5 chose red: angle = (5/20) x 360 = 90°
65
Box plots
A box plot shows: minimum, lower quartile (Q1), median (Q2), upper quartile (Q3), maximum. The box spans Q1 to Q3 (the interquartile range).
IQR = Q3 - Q1 (measures spread of middle 50%) Range = max - min
To compare two box plots: compare medians (average), compare IQR (consistency). Higher median = higher average. Smaller IQR = more consistent.
Use the specific values from the box plots in your comparison. "Group A has a higher median (52) than Group B (45)" scores better than just "Group A is higher".
187
Tree diagrams (independent events)
Each branch shows a possible outcome and its probability. Branches from the same point must sum to 1. For independent events, probabilities stay the same on each trial.
P(A and B) = P(A) x P(B) — multiply along branches P(A or B) = add the results of different paths
ExampleP(win) = 0.3 each game. P(win both) = 0.3 x 0.3 = 0.09. P(win exactly one) = 0.3 x 0.7 + 0.7 x 0.3 = 0.42
For "at least one" questions, it's often easier: P(at least 1) = 1 - P(none).
151
Conditional probability
When the outcome of one event affects the next (e.g. picking without replacement), probabilities change on the second pick.
P(A then B without replacement): e.g. 5 red, 3 blue → P(red then blue) = 5/8 x 3/7 = 15/56
The denominator decreases by 1 each pick. If the first pick matched, the numerator for that colour also decreases by 1.
Without replacement ≠ independent. The second set of branches has different probabilities from the first.

Ratio & proportion in data
When sharing amounts in a given ratio appears in a data context, apply the standard ratio method: find total parts, find value of one part, multiply.
Ratio a:b:c → total parts = a+b+c One part = total amount / (a+b+c)

Exam Technique — Paper 4H
This is a calculator paper. Use your calculator for every calculation, even simple ones, to avoid arithmetic errors. Show your method so you earn method marks even if you press the wrong button.
Check: is your answer reasonable? A person can't be 500 years old. A probability can't exceed 1. A length can't be negative.
Common errors: forgetting to square/cube in unit conversions; using the wrong bounds for max/min calculations; not stating circle theorem names; writing P(at least 1) = P(1) instead of 1 - P(0).
For "show that" questions, you must demonstrate every step clearly. The answer is given — the marks are for the working.
